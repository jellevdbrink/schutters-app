# Schutterstoernooi app

Front-end Angular app to make tournament information easily available to all participants. It is built for and used during the [schutterstoernooi](https://schutterstoernooi.nl) tournament, a yearly tournament hosted by NRC'22. 

To see the deployed version, click [here](https://app.schutterstoernooi.nl/).

## Features

* View upcoming and previous games, including scores.
* View group standings.
* Personalized home page, where users can select and save their team in order to quickly see their own games and standings.
* News page, automatically synced with the blog posts on the wordpress based website.
* A special page which can be displayed on a big screen with an automatic slideshow with current standings and sponsors.

## Install & run

The app should be used in combination with the official schutterstoernooi api.
